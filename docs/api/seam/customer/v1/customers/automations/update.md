# Update Customer Automation Configuration

- [Request Parameters](#request-parameters)
- [Response](#response)

Updates the automation configuration for a specific customer.
Creates a customer-level override if one does not already exist.


<details>

<summary>Authentication Methods</summary>

- Client session token

To learn more, see [Authentication](https://docs.seam.co/latest/api/authentication).
