# Automations

## Endpoints


[**`/seam/customer/v1/customers/automations/get`**](./get.md)

Gets the automation configuration for the authenticated customer.
Returns the merged configuration (customer overrides on top of workspace defaults).


[**`/seam/customer/v1/customers/automations/update`**](./update.md)

Updates the automation configuration for a specific customer.
Creates a customer-level override if one does not already exist.
