# Delete an Access Code

<mark style="color:red;">`DELETE`</mark> `https://connect.getseam.com/access_codes/delete`

## Request

### Request Parameters

| Name                                               | Type   | Description            |
| -------------------------------------------------- | ------ | ---------------------- |
| access\_code\_id<mark style="color:red;">\*</mark> | String | ID of the access code. |
