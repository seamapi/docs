---
title: 'Filtering Connect Webviews by Custom Metadata'
description: 'When listing Connect Webviews, you can filter by custom metadata.'
---

When you use [List Connect Webviews](https://docs.seam.co/latest/api/connect_webviews/list), you can filter the list by one or more [custom metadata](/core-concepts/connect-webviews/attaching-custom-data-to-the-connect-webview) pairs. Include the `custom_metadata_has` parameter with a JSON string that specifies the desired key:value pairs.

**Request:**

<CodeGroup>

```javascript JavaScript
const connect_webviews = await seam.connectWebviews.list({
  custom_metadata_has: {
    internal_account_id: 'user-1',
  },
})

console.log(connect_webviews)
```

```bash cURL
curl -X 'POST' \
  'https://connect.getseam.com/connect_webviews/list' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer ${API_KEY}' \
  -H 'Content-Type: application/json' \
  -d '{
  "custom_metadata_has": {
    "internal_account_id": "user-1"
  },
}'
```

```python Python
connect_webviews = seam.connect_webviews.list(
  custom_metadata_has = {
    "internal_account_id": "user-1"
  }
)

pprint(connect_webviews)
```

```ruby Ruby
connect_webviews = client.connect_webviews.list(
  custom_metadata_has: {
    "internal_account_id": "user-1"
  }
)

puts connect_webviews.inspect
```

```php PHP
$webviews = $seam->connect_webviews->list(
  custom_metadata_has: array('internal_account_id' => 'user-1')
);

echo json_encode($webviews);
```

```csharp C#
var customMetadata = new Dictionary<string, string>() {
  {"internal_account_id", "user-1"}
};

seam.ConnectWebviews.List(
  customMetadataHas: customMetadata
);
```
