## Overview

Omnitec manufactures electronic hotel locks and access control solutions designed for the hospitality industry. Their product line includes RFID-based door locks, elevator controllers, and energy management systems tailored for hotels and resorts.

Seam integrates with Omnitec online locks connected through the [Omnitec Rent&Pass Gateway](https://www.omnitecsystems.com/products/rent-and-pass/gateway-online) to provide remote lock and unlock actions, as well as online access code programming.

---

## Supported Devices

This integration supports Omnitec online locks that are connected via an [Omnitec Rent&Pass Gateway](https://www.omnitecsystems.com/products/rent-and-pass/gateway-online). The gateway enables real-time communication between Seam and your Omnitec locks over Wi-Fi.

<Warning>
  Seam only supports Omnitec locks that are online through a gateway. Offline-only Omnitec locks (without a gateway connection) are not supported.
</Warning>

For detailed information about the Omnitec devices that Seam supports, see our [Omnitec Supported Devices page](https://www.seam.co/manufacturers/omnitec).

---

## Supported Features

We support the following features:

- [Triggering web lock and unlock actions](/low-level-apis/smart-locks/lock-and-unlock)
- [Programming online access codes](/low-level-apis/smart-locks/access-codes) on locks that have a keypad

## Device Provider Key

To create a [Connect Webview](/core-concepts/connect-webviews) that enables your users to connect their Omnitec devices to Seam, include the `omnitec` device provider key in the `accepted_providers` list. For more information, see [Customize the Brands to Display in Your Connect Webviews](/core-concepts/connect-webviews/customizing-connect-webviews#customize-the-brands-to-display-in-your-connect-webviews).
