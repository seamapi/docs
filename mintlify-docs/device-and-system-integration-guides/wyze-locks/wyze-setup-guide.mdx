---
title: 'Wyze Setup Guide'
description: 'Connect your Wyze locks to Seam.'
---

To use Seam with Wyze locks, ask the device owner to complete the steps below. Locks must already be installed and connected to a Wyze Lock Gateway before proceeding.

### 1. Set up the Wyze app

1. Download and install the [Wyze app](https://support.wyze.com/hc/en-us/articles/360032748451-Wyze-App-Setup-Guide) and create an account if you don't have one.
2. In the Wyze app, add your Wyze devices.

### 2. Create a Wyze API key

1. Log in to the [Wyze developer portal](https://developer-api-console.wyze.com/#/apikey/view).
2. Click **Create an API key**.
3. Click **view** and note the API key.

### 3. Connect Wyze to Seam

1. In the form to connect Wyze to Seam, enter your Wyze app login credentials and the API key you created.
2. Click **Submit** to complete.

---

### Troubleshooting

If devices don't appear after connecting your account:

- Verify that each lock is online and connected to a Wyze Lock Gateway in the Wyze app.
- Confirm that you entered the correct API key from the Wyze developer portal.
