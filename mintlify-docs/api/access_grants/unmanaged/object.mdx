---
title: 'Unmanaged'
description: 'Use the unmanaged Access Grant endpoints to get, list, and update Access Grants that Seam tracks but does not manage (where is_managed is false).'
---

## Endpoints

[**`/access_grants/unmanaged/get`**](./get)

Get an unmanaged Access Grant (where is_managed = false).

[**`/access_grants/unmanaged/list`**](./list)

Gets unmanaged Access Grants (where is_managed = false).

[**`/access_grants/unmanaged/update`**](./update)

Updates an unmanaged Access Grant to make it managed.
