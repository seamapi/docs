---
title: 'Unmanaged'
description: 'Use the unmanaged access method endpoints to get and list access methods that Seam tracks but does not manage, usually filtered by Access Grant.'
---

## Endpoints

[**`/access_methods/unmanaged/get`**](./get)

Gets an unmanaged access method (where is_managed = false).

[**`/access_methods/unmanaged/list`**](./list)

Lists all unmanaged access methods (where is_managed = false), usually filtered by Access Grant.
