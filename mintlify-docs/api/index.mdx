## Granting Access

Use **Access Grants** to give people access to physical spaces. Access Grants are the recommended way to grant access on Seam — they work across standalone smart locks and access control systems, and automatically handle PIN codes, mobile keys, and key cards for you.

<CardGroup cols={2}>
  <Card title="Access Grants" icon="id-badge" href="/api/access_grants">
    Create, update, and revoke access grants. This is the primary API for granting access to any space.
  </Card>
  <Card title="Instant Keys" icon="bolt" href="/api/instant_keys">
    Issue temporary mobile keys that work without an app install.
  </Card>
</CardGroup>

<Info>
The [Access Codes](/api/access_codes) and [ACS Credentials](/api/acs/credentials) APIs are low-level alternatives. For most use cases, **Access Grants** handle everything automatically — creating the right credential type, programming it to the device, and managing the lifecycle.
</Info>
